# Enterprise-Networks-Automation

Deze repository is gemaakt voor het bewaren van xml bestanden en configs voor NETCONF.

De onderstaande bestanden zijn voor het testen van bepaalde delen.
- iosxe_01_base.xml
- iosxe_02_ip.xml
- iosxe_03_interfaces.xml
- iosxe_04_services.xml

iosxe_router_conf.xml is het groote bestand met verschillende versies 

deploy_netconf is het script ook met verschillende versies.

De basisconfiguraties gebruikt voor dit script staan ook in deze repository onder:
- Router-Config
- Switch-Config
